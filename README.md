# 🌍 WanderNest Travels - Website Project

## 📌 Project Overview
WanderNest Travels is a simple and responsive travel agency website designed as a college front-end web development project.  
It showcases travel destinations, tour packages, a gallery, and a contact form for users to explore and connect easily.

This project focuses on building a clean UI using HTML, CSS, and JavaScript.

---

## 🎯 Features

- 🏠 Home Page with hero section and featured destinations  
- 📖 About Page describing company mission and vision  
- ✈️ Experiences Page showing travel packages with prices  
- 🖼️ Gallery Page displaying travel images  
- 📞 Contact Page with working form validation  
- 📱 Responsive design (mobile-friendly layout)  
- 🍔 Hamburger menu for small screens  
- 🎨 Clean and modern UI design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
