# WanderNest Travels - Website Project

## 📌 Project Overview

WanderNest Travels is a simple and responsive travel agency website designed as a college front-end web development project.  

It showcases travel destinations, tour packages, a gallery slider, and a contact form for users to explore and connect easily.

This project focuses on building a clean UI using HTML, CSS, and JavaScript with a responsive layout and interactive features.

---

## 🎯 Features

- 🏠 Home Page with hero section (WanderNest Travels + Dream • Discover)
- 🍔 Left sidebar navigation menu (all pages linked)
- 📖 About Page describing mission, vision, and company details
- ✈️ Experiences Page showing travel packages with images and prices
- 🖼️ Gallery Page with automatic image slider (6 destinations)
- 📞 Contact Page with simple form validation and contact details
- 📱 Fully responsive design (mobile-friendly)
- 🎨 Clean UI suitable for college project presentation
- 🔄 Image slider with navigation arrows and dots
- ✨ Basic JavaScript interactivity (menu toggle, form validation)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
